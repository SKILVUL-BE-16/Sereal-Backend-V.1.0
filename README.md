## APIs Specification

### Users

- #### Register
- Method : POST
- Endpoint : /register
- Body :

```
{
    "name": String,
    "email": String,
    "password": String,
    "role": String
}
```

- Response

```
{
    "message": "data has been created!!"
}
```

- #### Login
- Method : POST
- Endpoint : /login

- Body :

```
{
    "email": String,
    "password": String
}
```

- Response :

```
{
    "message": "Anda berhasil login"
    "token",
}
```

- #### Kelas
- Method : GET
- Endpoint : /kelas
- Header: x-access-token = "token"
- Body :

```
{
     "name": String,
     "categories": categories.ObjectId,
     "status" Boolean
}
```

- Response :

```
{
     "message": "Success get all kelas",
     "data": [
       {
           "_id": String,
           "name": String,
           "categories": "categories.ObjectId",
           "status": Boolean
        }
     ]
  }
```
